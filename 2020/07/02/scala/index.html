什么是Scala

Scala是一门将面向对象和函数式编程结合在一起的高级编程语言

为什么学习Scala

  • Scala和java一样都属于JVM语言，流程与java类似，是先scalac编译成class字节码文件才能scala执行，而且Scala可以直接调用Java的类库，既继承了Java的良好的编程环境，又有自己独特的优点
  • Scala相对于Java可以更全面的面向对象编程（Java引用数据类型才能成对象，基本数据类型不能，但Scala所有数据类型都可都对象，即真正的万物皆对象）以及Java没有的函数式编程
  • 函数式编程使得Scala的语法更加简洁，更加高效
  • Scala相对于面对对象面向集合编程，而大数据的底层就是处理对象和集合，因此Scala对于学习大数据来说尤为重要
  • Spark框架是现有的非常强大的基于内存的快速、通用、可扩展的大数据分析计算引擎，并且学习大数据就需要用到Spark，而Spark就是基于Scala语言编程的，因此需要学习Scala

三，基础语法

一，变量和数据类型

  1. 注释

    scala注释和java一样

    单行注释 //

    多行注释 /* */

    文档注释 /** */

  2. 变量和常量

    常量标识符 val 变量标识符 var

    定义常量 ：val 常量名[:常量类型] = 初始值 val l：Int = 10 val l = 10

    定义变量 : var 变量名[:变量类型] = 初始值 var r: Int = 10 var r = 10

    • 注意：能用常量的地方不用变量
    • 注意：变量声明时，必须有初始值
    • 注意：var修饰的对象引用可以改变，val修饰的对象则不可以改变，但是对象的状态（值）确实可以改变 var变量可以new新对象，val常量不能再new新的对象
    • 注意：scala中如果一行只写一个语句,结尾的分号可以省略

  3. 标识符命名规则

    scala标识符命名规则和java类似，即用字母，数字，下划线和_,$，其中数字不能开头

    scala标识符与java标识符相比又增加了一些操作符（+，-，*，/，#，，！）等

    命名规范按照java规范即可，驼峰命名法之类的，见名知意

    • 注意：Scala语法中 方法前有def标识符

  4. 字符串

    Scala创建字符串方式有

    • 通过双引号包裹直接创建字符串常量变量：val xx = “kasa”

    • 通过new String的方法创建字符串：var xx = new String（参数）

    • 通过拼接创建：val xx = “hello”+”world”+10

    • 通过插值表达式创建：var xx = s”hello,10,${变量名/值/表达式}”

    • 通过方法创建：var xx = 10.toString 等等

    • 通过三引号的方式创建 ： “””….”””{三引号一般用于书写sql语句}

  5. Scala数据类型

    Scala类型全部面向对象

    Any：所有类的对象

    ​ AnyVal：值类型

    ​ Byte，Short，Int，Long，Float，Double，Boolean，Char

    ​ Unit：相当于java的void，有一个实例为（）

    ​ Stringops：不是一个具体的类，而是对java string的补充的一系列的统称，列如java String不能乘除，而 Scala String可以乘除

    ​ AnyRef：引用类型

    ​ String，java class/集合/数组，scala class/集合/数组….

    ​ Null：是所有的引用类型的子类，有一个实例null，null一般用于给引用数据类型赋值

    Nothing：所有类的子类，scala内部使用

  6. 数据类型转换

    scala数值类型与数值类型转换

    • 低精度转高精度：Int–>long 自动转换
    • 高精度转低精度：Double–>Int to数值类型方法

    scala数值类型转换

    • 数字转字符串：拼接/插值/toString
    • 字符串转数字：toXXX方法 var a = 10.toString
